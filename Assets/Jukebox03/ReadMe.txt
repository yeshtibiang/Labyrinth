Jukebox Tabletop #3

A detailed classic Jukebox model in 2 styles: 
clean and dirty.

LODs are included and a second UV map is added for lightmapping.
The main textures are 2048x2048 in size
(glass texture 512x512) and are PBR.

Tri count
LOD0		1134 tris
LOD1		856 tris
LOD2		658 tris
